## 기획 배경
### 개요
<h3> 여름철 폭우로 인해 전국적으로 침수 피해가 끊이지 않고 있다. 앞으로 '장마'가 아닌 '우기'로 단어를 변경해야 한다는 주장도 나오고 있는 가운데, 정부에서도 침수 예방 대책을 구축, 개선 하려고 한다. </h3>
